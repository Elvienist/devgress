# BYOD Registration and Monitoring System
Desktop-based JavaFX application for managing student
device ingress and egress on campus.

## Tech Stack
- JavaFX
- PostgreSQL hosted on Supabase
- JDBC
- Maven

## Setup
1. Clone the repository
2. Copy config/config.properties.example to
   src/main/resources/config.properties
3. Fill in your Supabase connection credentials
4. Run the project in IntelliJ
